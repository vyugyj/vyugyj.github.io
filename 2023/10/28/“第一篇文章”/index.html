C语言

第一章 程序设计概述

1.1

  • 三位著名科学家
    • 艾伦 ·马西森·图灵：计算机科学之父，人工智能之父，提出了著名的“图灵机”，提出了“图灵机”和”图灵测试”等重要概念。
    • 约翰·冯·诺伊曼：提出了”存储程序“设计思想和二进制原理。
    • 克劳德·埃尔伍德·香农：消除信息的不确定性，被誉为“信息论及数字通信之父”。
  • 现代计算机系统是由紧密相关的硬件（固件）和软件组成的一个复杂整体。
  • 机器语言（二进制语言）是计算机唯一能识别和执行的语言。
  • 1Byte(字节)=8bit
  • 1K=2^10=1024Byte

1.2

  • 位,字节和地址是计算机最基本的工作单位。
  • C语言允许直接访问物理地址，因而它可以直接对硬件进行操作。

1.4

  • 结构特点

    • 头文件包括stdio.h和stdlib.h

    • main()是主函数，每个C语言程序都只能有一个main()主函数，main()函数可以放在程序首部，中间以及尾部

    • 如果函数没有返回值，那么声明为void类型，即“空类型”

    • printf()函数是格式化输出函数

    • \n是转义字符，其作用是实现换行

    • 函数体通过一组花括号“{}”括起来，每个语句后使用一个分号“;”表示语句结束

    • /······/和//分别表示段落注释和行注释

    • printf()函数的函数原型在stdio.h中，system()函数的函数原型在stdlib.h中，因此在使用这两个函数前需要先包含这两个文件

  • 符号

    • int用于定义整数类型变量的标识符

    • “%d”是格式化字符，表示十进制有符号整数

    • scanf()是C语言中的一个终端格式化输入函数

      scanf(“%d,%d”,&a,&b) &取地址符号

1.数制及其转换

  • 十进制转换二（八,十六)进制

    • 整数部分：除2（8,16）取余，结果倒序

    • 小数部分：乘2（8,16）取整，结果正序

  • 8421

    正数 原码，反码和补码三者一致
    负数 原码变成反码:符号位不变，其余各位取反：反码变补码：+1；补码变原码：符号位不变，其余各位取反+1

  • 在计算机中，所有数均以补码存放

3.2

  • 标识符由字母，数字和下划线组成，且第一个字符必须是字母或下划线

  • 用户自定义标识符要避免与C语言的关键字（保留字）冲突

  • #define 标识符 常量

    符号常量中标识符一般用大写形式

  • const 数据类型 常变量名=初始值

  • 变量名表可以是一个变量，也可以是用逗号分隔的多个变量
